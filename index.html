Incremental Mean Update

The elegant recurrence that underpins every sample-average value estimator in RL — from k-armed bandits to Q-learning. Derived from Sutton & Barto, Reinforcement Learning, Ch. 2.

INCREMENTAL UPDATE RULE
Qn+1 = Qn  +  (1/n)  ·  [Rn − Qn]
error = Rn − Qn  (prediction error)
α = 1/n  → shrinks with experience
Equivalent to exact mean — zero extra memory.
O(1) Memory
Step-by-step derivation
Start with exact mean
Qn+1 = (1/n) Σni=1 Ri
Definition: average of all n rewards seen so far.
Peel off latest reward
= (1/n) · (Rn + Σn−1i=1 Ri)
Separate the newest reward from the historical sum.
Recognise Qn (the circle!)
= (1/n) · (Rn + (n−1) · Qn)
The sum over (n−1) terms equals (n−1)·Qn by definition.
Expand & collect
= (1/n)(Rn + nQn − Qn)
Distribute (n−1) = n − 1 into the bracket.
Final form ✓
Qn+1 = Qn + (1/n)[Rn − Qn]
Classic NewEstimate ← OldEstimate + α·Error structure.
Non-stationary variant
Qn+1 = Qn + α[Rn − Qn]
Fix α ∈ (0,1] to weight recent rewards exponentially more.

# ── Incremental Mean Update — Sutton & Barto Ch.2 ──────────────────────

class IncrementalMean:
    """Sample-average action value estimator (O(1) memory)."""
    def __init__(self, Q0=0.0):
        self.Q = Q0   # current estimate  →  Q_n
        self.n = 0    # step counter

    def update(self, R):
        """Feed one reward; returns (new_Q, error, alpha)."""
        self.n += 1
        alpha  = 1.0 / self.n             # step-size α = 1/n
        error  = R - self.Q               # prediction error δ
        self.Q += alpha * error            # Q ← Q + α·δ
        return self.Q, error, alpha


class IncrementalMeanConstantAlpha:
    """Non-stationary variant: fixed α weights recent rewards more."""
    def __init__(self, alpha=0.1, Q0=0.0):
        self.Q     = Q0
        self.alpha = alpha
        self.n     = 0

    def update(self, R):
        self.n += 1
        error  = R - self.Q
        self.Q += self.alpha * error
        return self.Q, error, self.alpha


# ── Demo ────────────────────────────────────────────────────────────────
if __name__ == "__main__":
    rewards = [4, 2, 7, 1, 5, 3, 8, 6]
    tracker = IncrementalMean()

    print(f"{'Step':>5} {'Reward':>8} {'Error':>10} {'Alpha':>8} {'Q_est':>12}")
    print("-" * 48)
    for R in rewards:
        Q_new, err, alpha = tracker.update(R)
        print(f"{tracker.n:>5} {R:>8.2f} {err:>+10.4f} {alpha:>8.4f} {Q_new:>12.6f}")
